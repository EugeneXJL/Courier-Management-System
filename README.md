# Courier Management System

## Project Overview
This project is a Courier Management System designed to support shipment creation, package tracking, and delivery management.

Users can create shipment requests and track packages, while administrators can manage packages and update delivery status.

The system follows the CRUD (Create, Read, Update, Delete) model.

---

## Features

User functions
- Create shipment
- Track package
- View shipment history

Administrator functions
- View package records
- Update delivery status
- Delete package records
